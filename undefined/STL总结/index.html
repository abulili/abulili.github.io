STL总结

排序： sort(v.begin(), v.end()) （包括字母）（less<int>() / greater<int>()）

vector

vector<int> a(n + 1); 能先开辟先开辟，如果输入失败又可能这里出问题了

vector<int> b = a; 定义的时候等于，否则可能会出问题

map

默认key从小到大排序

自定义按value :sort cmp(pair<int, int>a, pair<int, int>b) return a.second < b.second;

获取最大值： auto i = max_element(t.begin(), t.end(), cmp) (还有min_element)

map<int, set<string>> m;

word[len] = set<string>(); (只是创建空间， 会重置原数据)

返回出现次数：m.count(length)

可以mp.first(); mp.second();

pair

先比较第一个，第一个相同再比较第二个

当想插入两个： q.push(make_pair(0, 0));

string

大小转 小写：transform(s.begin(), s.end(), s.begin(),tolower)

小写转大写：toupper

有空格：getline(cin, s);

s.substr(0，n): 从1开始数n个

s.reverse(s.begin(), s.end()) : 字符串翻转

set

去重 + 自动排序

multiset： 不去重（可以删除某个点）

set.count(string s) ： set内有几个与s相同的 //string同样可用
